# Prompt Library - [Tên môn học]

> Các prompt tái sử dụng cho AI khi làm việc với môn này.

---

## Tóm tắt bài giảng

```
Tóm tắt nội dung bài giảng sau thành các key points. 
Giữ lại frameworks, models, definitions quan trọng.
Viết bằng tiếng Việt, ngắn gọn, bullet points.

[Paste nội dung]
```

## Phân tích case study

```
Phân tích case study sau theo framework [SWOT/5 Forces/PESTEL/...]:
- Company background
- Problem identification  
- Analysis using framework
- Recommendations
- Key lessons

[Paste case]
```

## Review bài viết

```
Review bài viết sau theo rubric:
- [Tiêu chí 1]: ...
- [Tiêu chí 2]: ...
- [Tiêu chí 3]: ...

Đánh giá từng tiêu chí, cho điểm gợi ý, và suggest cải thiện cụ thể.

[Paste bài viết]
```

## Giải thích concept

```
Giải thích concept "[tên concept]" trong context [môn học].
- Definition
- Key components
- Real-world examples (VN context nếu có)
- Liên hệ với các concept khác trong môn
```

## Tạo flashcards

```
Tạo 20 flashcards cho chủ đề [topic] dưới dạng Q&A.
Focus vào definitions, frameworks, và applications.
Format: Q: ... | A: ...
```
